# Product Manager (Console App)

A console-based product management system built using C#.

## Features

- Add products
- View all products
- Search product (LINQ)
- Delete product (safe removal)
- Product summary (GroupBy with LINQ)
- Input validation
- Reusable UI display methods

## Tech Used

- C#
- LINQ
- Object-Oriented Programming

## Purpose

Practice backend architecture, LINQ operations, and clean code design.
